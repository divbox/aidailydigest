# AI Digest Hub

A daily AI newsletter digest, generated from Gmail newsletters by a Claude Cowork skill and auto-published here.

**[index.html](index.html)** always shows the latest digest — this is the link to bookmark or pin.

**[archive.html](archive.html)** lists every past digest, grouped by month.

New digests are generated most weekday mornings. Publishing (regenerating the index/archive and pushing here) runs automatically on the maintainer's machine — no manual steps day to day.
